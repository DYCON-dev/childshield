# ChildShield Browser Extension

> Planned for V2.

A WebExtension for Chrome, Firefox, and Safari that lets you drag a photo into the browser and get a blurred copy — without uploading anything.

## Planned stack

- WebExtension API (works on Chrome, Firefox, and Safari 14+)
- [face-api.js](https://github.com/justadudewhohacks/face-api.js) — face detection + age estimation, MIT, ~6 MB total bundled with the extension
- Vanilla JS / TypeScript (TBD)
