# ChildShield Safari Extension

> Planned for V2.

macOS Safari wrapper around the [ChildShield WebExtension](../extension). Safari 14+ supports the WebExtension API, so the JavaScript code is identical — this Xcode project only provides the App Extension bundle required to ship on the App Store.
